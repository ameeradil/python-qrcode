# Python QR Code Generator

A simple Python script that generates a QR code from user input and saves it as an SVG file.

## 📌 Features

* Takes URL input from the user
* Generates a QR code instantly
* Saves output as `.svg` file

## 🚀 Installation

Make sure you have Python installed, then install the required package:

```bash
pip install "qrcode[svg]"
```

## 🧑‍💻 Usage

Run the script:

```bash
python3 qr.py
```

You will be prompted to enter a URL:

```
Enter your URL: https://example.com
```

## 📂 Output

After running the script, a file will be created:

```
qrcode.svg
```

## 🛠 Requirements

* Python 3.x
* qrcode library with SVG support
